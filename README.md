# TinyTales UI Page

This project is a **single UI page** built for the **TinyTales** brand as part of a **job task**.  
The goal is to accurately implement a provided **UI mockup** using modern frontend tools.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Purpose

- Recreate one UI page based on a design mockup
- Focus on layout accuracy and visual consistency
- Use clean, reusable components
- Apply subtle UI animations

> This task is **UI-only**. No backend or API integration is included.
